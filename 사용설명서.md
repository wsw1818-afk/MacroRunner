# VBA 매크로 관리자 (Macro Runner) 사용 설명서

## 프로그램 목적

보안 정책으로 VBA 매크로가 포함된 파일을 저장할 수 없는 환경에서:
- 매크로 코드를 외부 파일로 저장/관리
- 열려있는 Excel/PowerPoint에 매크로를 주입하여 실행
- 파일 저장 없이 메모리에서만 동작

---

## 설치 방법

### 방법 1: 빌드된 프로그램 사용 (권장)

1. `build.bat` 더블클릭하여 빌드
2. `dist/MacroRunner/` 폴더 전체를 원하는 위치에 복사
3. `MacroRunner.exe` 실행

### 방법 2: Python으로 직접 실행

```bash
# 의존성 설치
pip install pywin32

# 실행
python macro_runner.py
```

---

## 폴더 구조

### 빌드 후 배포 폴더

```
dist/MacroRunner/           <- 이 폴더 전체를 배포
├── MacroRunner.exe         <- 실행 파일
├── _internal/              <- 필수 라이브러리 (삭제 금지!)
└── macros/                 <- 매크로 저장 폴더
    └── macro_index.json    <- 모든 매크로가 저장된 파일
```

### 개발 폴더

```
MacroRunner/
├── macro_runner.py         <- 메인 소스코드
├── build.py                <- 빌드 스크립트
├── build.bat               <- 빌드 실행
├── macros/                 <- 매크로 저장 폴더
│   └── macro_index.json
└── 사용설명서.md
```

---

## 배포 방법

### 다른 사람에게 공유하기

1. `build.bat` 실행하여 빌드
2. `dist/MacroRunner/` 폴더 전체를 **ZIP으로 압축**
3. ZIP 파일 전달
4. 받는 사람: 압축 해제 후 `MacroRunner.exe` 실행

**주의:** `MacroRunner.exe` 파일만 주면 안 됩니다! `_internal/` 폴더가 반드시 함께 있어야 합니다.

### 매크로만 공유하기

이미 프로그램이 있는 사람에게는:
- `macros/macro_index.json` 파일만 전달
- 기존 파일을 교체하면 매크로가 업데이트됨

## 사전 설정 (중요!)

### Excel/PowerPoint 보안 설정

매크로 주입을 위해 VBA 프로젝트 접근 권한이 필요합니다:

**Excel:**
1. 파일 → 옵션 → 보안 센터 → 보안 센터 설정
2. "매크로 설정" 탭 선택
3. "VBA 프로젝트 개체 모델에 대한 액세스 신뢰" 체크

**PowerPoint:** 동일한 방법으로 설정

> 이 설정이 없으면 "VBA 프로젝트 접근 권한이 없습니다" 오류 발생

## 사용 방법

### 기본 흐름

```
1. Excel 또는 PowerPoint 실행 (새 문서 열기)
2. MacroRunner.exe 실행
3. 대상 프로그램 선택 (Excel / PowerPoint)
4. 매크로 선택 또는 직접 입력
5. "주입 & 실행" 버튼 클릭
```

### 매크로 저장하기

1. "새 매크로" 버튼 클릭
2. 이름, 설명 입력
3. 코드 영역에 VBA 코드 작성
4. "저장" 버튼 클릭

### 파일에서 불러오기

- `.vba`, `.bas`, `.txt` 파일 지원
- "파일에서 불러오기" 버튼으로 코드 가져오기

### 클립보드 사용

- 웹이나 다른 문서에서 VBA 코드 복사
- "클립보드에서 붙여넣기" 버튼 클릭

### 매크로 실행

- **주입 & 실행**: 코드를 주입하고 지정한 Sub/Function 실행
- **주입만**: 코드만 주입 (Alt+F8로 수동 실행)

## 주의사항

1. **Office가 먼저 실행되어 있어야 함**
   - Excel이나 PowerPoint를 먼저 열고, 그 다음 MacroRunner 실행

2. **문서가 열려 있어야 함**
   - 빈 새 문서라도 열려 있어야 매크로 주입 가능

3. **저장은 안 됨**
   - 보안 정책 때문에 매크로가 포함된 문서 저장은 여전히 불가
   - 매크로는 메모리에서만 실행됨

4. **신뢰할 수 있는 매크로만 사용**
   - 출처가 불분명한 매크로는 실행하지 마세요

## 문제 해결

### "Excel/PowerPoint 실행 안 됨"
- Office 프로그램을 먼저 실행하세요

### "열린 통합 문서가 없습니다"
- 새 문서나 기존 문서를 열어주세요

### "VBA 프로젝트 접근 권한이 없습니다"
- 위의 "사전 설정" 항목 참조
- 파일 → 옵션 → 보안 센터에서 설정 변경

### PowerPoint 매크로가 자동 실행 안 됨
- PowerPoint는 보안상 자동 실행이 제한됨
- "모듈 삽입 완료" 메시지 후 Alt+F8로 수동 실행

## 포함된 샘플 매크로

### Excel용
| 이름 | 기능 |
|-----|------|
| 샘플_데이터정리 | 빈 행 삭제, 중복 제거, 정렬 |
| 샘플_서식변경 | 테이블 스타일, 조건부서식, 숫자형식 |

### PowerPoint용
| 이름 | 기능 |
|-----|------|
| 샘플_슬라이드정리 | 폰트 통일, 제목 크기, 텍스트 정렬 |

## 팁

1. **자주 쓰는 매크로는 저장해두기**
   - 한 번 저장하면 계속 재사용 가능

2. **Sub Main() 규칙 사용**
   - 메인 실행 함수 이름을 `Main`으로 통일하면 편리

3. **더블클릭으로 빠른 실행**
   - 매크로 목록에서 더블클릭하면 바로 실행

4. **백업은 간단!**
   - `macros/macro_index.json` 파일만 복사하면 모든 매크로 백업
